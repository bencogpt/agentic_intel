1. טענות לגבי לבנון שאינן מגובות בעדויות עדכניות
2. הנחות שגויות לגבי מאזן הכוחות הפנים-לבנוני
3. עדויות סותרות לטענות המרכזיות
4. חלופות פרשנות שלא נשקלו

עבוד בעברית. ציין מקורות לכל ממצא.
